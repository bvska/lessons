Дан класс User со следующими свойствами:
String имя,
LocalDate дата рождения,
enum пол,
String город,
int сколько всего потратил на покупки,
enum роль (постоянный покупатель, редко покупает, ничего не купил)

 Дан список пользователей List<User>

Задачи:
1. Найти средний возраст покупателей
2. На основе списка собрать мапу, где ключ - пол, значения - сумма потраченных пользователями денег
3. На основе списка собрать мапу: ключ - город, значение - список объектов соответствующих объектов
4. На основе списка собрать мапу: ключ - роль, значение - список объектов соответствующих объектов
5. На основе списка собрать мапу: ключ - год рождения, значение - список объектов соответствующих объектов
6. Отсортировать пользователей по дате рождения
7. На основе списка создать новый список, в который войдут пользователи определенной роли
